## 1. 相对估值

将估值倍数与同行和历史平均值比较。

| 倍数 | 最适用场景 | 巴菲特看法 |
|------|----------|-----------|
| P/E（市盈率） | 成熟盈利公司 | 有用但可被会计手法操纵 |
| P/S（市销率） | 高增长/未盈利公司 | "收入是虚荣，利润是理智" |
| P/B（市净率） | 金融/重资产行业 | 早期投资中常用 |
| EV/EBITDA | 跨资本结构比较 | 对高杠杆行业优于P/E |

比较维度：自身5年历史平均 → 直接竞争对手 → 行业平均 → 历史分位数

## 2. DCF现金流折现

内在价值估算的黄金标准。

**步骤：**
1. 基于收入增长、利润率、CapEx假设，预测5-10年FCF
2. 计算终值（退出倍数法 + 永续增长法交叉验证）
3. 用WACC折现所有现金流
4. 加总→企业价值→减债务加现金→股权价值

**敏感性分析维度：** 收入增长率（±5ppts）、营业利润率、WACC（±1-2ppts）、终端增长率（2-3%）

## 3. 所有者盈余收益率

巴菲特最偏好。

```
所有者盈余 = 净利润 + 折旧摊销 + 非现金费用 - 维护性CapEx ± 营运资金变动
所有者盈余收益率 = 所有者盈余 / 总市值
```

解读：>8%很有吸引力 | 5-8%合理 | 3-5%偏贵 | <3%很可能高估

## 4. PEG比率

```
PEG = 市盈率 / 预期EPS增长率
```

解读：<1.0可能低估 | 1.0-1.5合理 | 1.5-2.0偏贵 | >2.0需要非常出色的增长

## 5. DDM股利折现

适用于高分红、稳定股利的公司（如公用事业、银行、消费龙头）。

```
内在价值 = D₁ / (r - g)

D₁ = 下一年预期股利
r = 要求回报率（股权成本）
g = 股利可持续增长率
```

**注意：** g不应超过GDP长期增长率（通常2-4%）。对于不分红或分红不稳定的公司，此方法不适用。

## 6. NAV净资产价值

适用于地产、金融、控股公司。

**步骤：**
1. 对资产负债表上的资产逐项重估（按市场价而非账面价值）
2. 减去全部负债
3. 得到重估净资产
4. 计算NAV折价/溢价 = (股价 - 每股NAV) / 每股NAV

**应用：** 地产股常见30-50%NAV折价；低于此区间可能有投资机会。

## 7. 反向DCF

不是向前预测，而是反推："当前价格隐含了多高的增长率？"

**步骤：**
1. 取当前股价
2. 反向运行DCF模型
3. 求解隐含的收入/盈利增长率
4. 追问："这个增长率现实吗？"

**关键应用：** 如果市场隐含10年40%年增速而公司历史仅15%，则股价为完美定价；如果隐含仅5%而实际20%，可能有机会。

## 8. SOTP分部加总

适用于多元化集团或有多个独立业务的公司。

**步骤：**
1. 识别每个独立业务板块
2. 对每个板块独立估值（使用最适合的方法）
3. 加总各板块价值
4. 评估是否需要集团折价（通常10-20%）
5. 与当前市值比较

**典型案例：** Alphabet = 搜索广告（P/E估值）+ Google Cloud（P/S估值）+ Waymo（可比交易）+ 其他赌注（风投估值）
