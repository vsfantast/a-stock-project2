## 四道过滤器

巴菲特反复描述，他让每个投资机会通过四道过滤器：

### 过滤器1：我能理解它吗？
- 生意必须简单且可预测
- 必须能以合理的信心预测未来现金流
- "永远不要投资你无法理解的生意"
- 如果需要博士学位才能理解收入模型，就跳过

### 过滤器2：它有持久的竞争优势吗？
- "护城河"——是什么阻止竞争对手侵蚀利润？
- 护城河必须随时间推移而加宽，不是变窄
- "在商业中，我寻找被坚不可摧的护城河保护的经济城堡"
- 最好的护城河类型：品牌（可口可乐）、转换成本（苹果）、网络效应（Visa）、成本领先（GEICO）

### 过滤器3：管理层是否诚实且有才能？
- 诚信不可谈判——"我们承受得起亏钱，甚至亏很多钱。但承受不起丧失声誉"
- 资本配置能力最重要——他们如何运用现金？
- 寻找像老板一样思考的管理者，而非打工心态
- 危险信号：建造帝国、过度福利、把糟糕结果怪罪外部因素

### 过滤器4：价格合理吗？
- 即使最好的企业，在错误价格下也是糟糕的投资
- 内在价值 = 所有未来现金流的现值
- 要求安全边际——价格和价值之间的差距
- "以合理的价格买一家杰出的公司，远好于以杰出的价格买一家平庸的公司"

## 关键财务指标偏好

### 所有者盈余（非GAAP利润）
巴菲特最偏好的指标。不像GAAP净利润，所有者盈余反映所有者能从企业中实际提取的现金。

计算公式：
```
所有者盈余 = 净利润 + 折旧摊销 + 其他非现金费用 - 维护性资本开支 ± 营运资金变动
```

### 股本回报率（ROE）
- 目标：持续多年 > 15%
- 必须在没有过度杠杆的情况下实现
- 高ROE + 低负债 = 卓越企业

### 负债水平
- 偏好能在3-4年盈利内偿还所有债务的公司
- 长期负债 / 所有者盈余 < 4x
- "我见过更多人因酒精和杠杆而失败"

### 自由现金流
- FCF应持续超过净利润（盈利质量标志）
- "盈利"公司出现负FCF是危险信号
- FCF转化率 > 100% 为理想状态

### 利润率
- 持续或扩张的利润率 = 定价权 = 护城河证据
- 收缩的利润率 = 竞争压力 = 护城河侵蚀
- 重要的是相对同行的利润率地位

## 巴菲特回避清单

1. **不理解的生意** — "投资中没有被判三振"
2. **负债过多的公司** — "只有退潮时，你才能发现谁在裸泳"
3. **缺乏诚信的管理层** — "招人看三样：诚信、聪明和精力。没有第一样，后两样会害死你"
4. **转型中的困难企业** — "转型很少能成功"
5. **IPO** — "IPO就像协商交易——卖方选择的时间不会对你有利"
6. **热门行业** — "投资的关键是判断竞争优势和其持久性"

## 估值哲学

### 内在价值
> "内在价值可以简单定义：它是企业在其剩余生命中能够取出的现金的折现值。"

### 两栏法
巴菲特和芒格用"你放弃什么 vs 你得到什么"来思考：
- **A栏**：我在支付什么？（市值、企业价值）
- **B栏**：我得到什么？（所有者盈余、增长率、护城河持久性、管理层质量）

### 所有者盈余收益率解读
- \>8% = 非常有吸引力（可能被低估）
- 5-8% = 优质企业的合理价值区间
- 3-5% = 偏贵，需要高增长支撑
- <3% = 很可能高估

## 买卖时机判断

### 何时买入
- 当企业杰出且价格提供安全边际时
- "在别人贪婪时恐惧，在别人恐惧时贪婪"
- 最佳买入机会来自市场恐慌、行业低谷或暂时性困难

### 何时卖出
- "我们最喜欢的持有期是永远"
- 只在以下情况卖出：
  1. 护城河被永久损害
  2. 管理层丧失诚信
  3. 你在原始分析中犯了错
  4. 你找到了显著更好的标的
