## 🔮 巴菲特的裁决

## 模块3：商业模式深度解析

### 收入模型

腾讯的收入分为四大板块：

**1. 游戏（约32%）**
- 国内游戏：约1500亿（同比增长10%）
- 国际游戏：约774亿（同比增长33%）
- 代表作：《王者荣耀》《和平精英》《PUBG Mobile》《Supercell》

**2. 社交网络（约25%）**
- 微信/QQ会员费、视频号订阅
- 音乐、流媒体订阅
- 收入增速放缓，占比下降

**3. 金融科技与企业服务（约38%）**
- 微信支付（手续费）
- 云服务（腾讯云）
- 商业支付（线下扫码）
- 增速最快，但利润率偏低（云业务）

**4. 广告（约20%）**
- 朋友圈广告
- 视频号广告（最大增量）
- 2025年广告收入同比增长20%+，AI定向投放效果显著

### 一句话测试

> "腾讯是一家通过微信生态牢牢锁定10亿用户，然后向他们推送游戏、广告、金融服务来变现的互联网平台公司。"

**单位经济**：
- 微信用户ARPU：约¥400-500/年（含支付、游戏、广告全部贡献）
- 游戏付费用户ARPPU：约¥300-500/月
- 金融科技毛利率：约45-50%（支付业务）
- 云业务毛利率：约20-25%（逐步改善）

### 产业链定位

```
上游：内容提供商（游戏开发商、电影/音乐版权）
     ↓
平台：微信（流量分发）、云（基础设施）
     ↓
变现：游戏（现金牛）、广告（增长）、金融（平台费）
     ↓
用户：10亿+用户（微信）
```

**议价能力**：
- 对用户：极强（微信不可替代）
- 对广告主：强（但受经济周期影响）
- 对游戏开发商：中等（CP有谈判空间）
- 对支付商户：中等（支付宝竞争）

## 模块4：经济护城河分析

| 护城河来源 | 评级 | 证据 |
|-----------|------|------|
| **网络效应** | 极宽 | 微信10亿用户→所有沟通必须用它→不可能迁移。边际用户越多，生态越强 |
| **转换成本** | 极高 | 换掉微信=换掉所有社会关系，13亿人不可能同时换 |
| **成本优势** | 高 | 规模效应（服务器、带宽、内容成本均摊） |
| **无形资产** | 高 | 微信品牌、QQ品牌、数千款游戏版权 |
| **有效规模** | 高 | 即时通讯只需一个，腾讯已占据 |

**总评**：极宽护城河（互联网最高等级），且稳定。

**护城河趋势**：稳定（微信生态依然稳固，但面临抖音等短视频侵蚀时间）。

**10年后护城河展望**：微信可能依然是主力通信工具（类似现在的email），但人们的注意力会被短视频（抖音/视频号）瓜分。**核心风险**：短视频是否最终替代文字/语音通信？目前看答案是否定的，但需持续观察。

| 利息覆盖倍数 | >20x | 优秀 |

**所有者盈余计算**：
```
净利润：2,248亿
+ 折旧与摊销：约150亿
+ 其他非现金费用：约80亿
- 维护性资本开支：约240亿（估算600亿 CapEx 的40%）
± 营运资金变动：约-100亿

所有者盈余 ≈ 2,248 + 150 + 80 - 240 - 100 ≈ 2,138亿
```

### 杜邦分析（ROE拆解）

```
ROE = 净利率 × 资产周转率 × 财务杠杆
    = 30% × 0.50 × 1.67
    = 28%
```
