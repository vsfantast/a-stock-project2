## 模块3：商业模式深度解析

### 收入模型

宁德时代的主营业务分为三大板块：

**1. 动力电池系统（约占60%）**
- 为新能源汽车提供电芯、模组、电池包
- 客户：特斯拉、蔚来、理想、小鹏、宝马、奔驰等
- 收入确认：按合同交付确认，收入结构偏项目制

**2. 储能系统（约占20%）**
- 电网储能、工商业储能、家庭储能
- 政策驱动：中国"双碳"目标、欧洲、美国 IRA 法案
- 增速最快板块，2025年收入同比增长约100%

**3. 电池材料与回收（约占15%）**
- 正极材料、负极材料、电解液
- 电池回收业务（闭环经济）
- 毛利率偏低，约15-18%

**4. 其他（ licensing、 service，约5%）**

### 一句话测试

> "宁德时代是一家电池制造商，通过向全球新能源汽车和储能系统制造商销售动力电池和储能产品，同时向上游材料端延伸以控制成本来赚钱。"

**单位经济**：
- 平均单车电池价值量：约¥50,000-80,000（取决于车型）
- 毛利率：26%（行业平均水平约12-15%）
- 账期：2-3个月，较为健康
- 重复采购：车厂一旦绑定，切换成本高（3-5年合同周期）

### 产业链定位

```
上游：锂矿、钴矿、镍矿（锂价波动为核心风险）
     ↓
中游：正极材料、负极材料、隔膜、电解液（宁德时代部分自产）
     ↓
下游：新能源汽车制造、储能系统集成商
     ↓
终端：消费者（个人购车）、企业（运营车队）
```

**议价能力**：
- 对上游：作为全球最大买家，有较强议价能力（但锂矿价格不可控）
- 对下游：技术领先，客户粘性强，但面对特斯拉等大客户时议价能力相对较弱

## 模块4：经济护城河分析

| 护城河来源 | 评级 | 证据 |
|-----------|------|------|
| **网络效应** | 中 | 客户多→规模大→成本低→更多客户，但电池制造业网络效应不如平台公司强 |
| **转换成本** | 高 | 整车厂切换电池供应商需2-3年重新认证，测试周期长 |
| **成本优势** | 高 | 规模全球第一，CTP技术简化工艺，单位成本比竞争对手低10-15% |
| **无形资产** | 中 | 技术专利（麒麟电池、CTP）、客户信任，但品牌本身不如消费品强 |
| **有效规模** | 高 | 动力电池行业规模效应极强，CR3合计占比约70%，宁德时代独居龙头 |

**总评**：宽护城河（制造业标准），且在持续加宽。

**护城河趋势**：加宽中。证据：
- CTP 3.0（麒麟电池）技术领先对手至少1-2代
- 海外工厂布局（匈牙利、德国），全球化护城河延伸
- 储能业务爆发，新增长曲线确认

**10年后护城河展望**：如果固态电池/钠离子电池技术路线没有颠覆性变化，宁德时代的领先地位将持续。**最大风险**：技术路线颠覆（固态电池量产成本大幅下降）。
