## 模块3：商业模式深度解析

### 收入模型

Circle的收入几乎完全依赖于USDC的储备利息：

**核心业务逻辑**：
```
用户持有USDC → Circle将USD储备投资于美国国债/银行存款 → 赚取利息 → 返还给用户（部分）→ Circle留取差价
```

**实际收入结构（2025年）**：
- **储备利息收入**：约$2.1亿/季度（全年约$8亿），占总收入~80%
- **交易手续费**：约$0.3亿/季度（链上交易量近$12万亿）
- **其他服务**（企业级USDC API、BaaS）：约$0.2亿/季度
- **EURC（欧元稳定币）**：贡献较小，约5%

**USDC流通量与收入的关系**：
- 2025年底USDC流通量：$753亿
- 对应储备规模：约$760亿（略高于流通量，有浮动）
- 年化收益率：约4.5%（美联储加息周期）
- **年收入贡献**：约$34亿（按4.5%计）→ 但实际Circle只留取其中约25%，其余返还给用户

### 一句话测试

> "Circle是一家稳定币发行商，通过将用户持有的USDC储备金投资于美国国债赚取利息差来赚钱，本质上是一家'数字时代的银行'，但受制于监管政策。"

**单位经济**：
- 每1亿美元USDC流通 → 每年带来约$450万利息收入
- Circle从中留取约$100-120万（利润率约25-30%）
- 其余作为"持有者奖励"返还，维持USDC的竞争力

**产业链定位**：
```
上游：美国国债、联邦存款保险公司（FDIC）担保的银行存款
     ↓
中游：Circle（稳定币发行、储备管理、合规运营）
     ↓
下游：USDC用户（个人、机构、企业）、DeFi协议、NFT交易、跨境汇款
```

**关键风险**：监管禁止"生息"稳定币 → 收入归零。
