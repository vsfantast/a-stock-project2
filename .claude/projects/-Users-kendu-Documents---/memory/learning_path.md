# 🗺️ 学习路线

---

## 🟢 第一阶段：活下来（第 1 个月）

| 目标 | 内容 |
|:------|:------|
| 不亏钱 | 止损纪律、仓位管理 |
| 建习惯 | 每天复盘 15 分钟、写交易日志 |
| 懂术语 | PE / PB / ROE / MACD / RSI / 换手率 |

**检验标准：** 一个月后能看懂分析报告，知道每句话在说什么。

---

## 🟡 第二阶段：建框架（1-3 个月）

| 目标 | 内容 |
|:------|:------|
| 会分析 | 能独立做三家同行业对比 |
| 会看线 | 均线 / 量价 / MACD 金叉死叉 |
| 会估值 | DCF 三情景 + PEG 估值的含义 |

**检验标准：** 自己选的股票能写出 500 字分析，逻辑自洽。

---

## 🔴 第三阶段：独立决策（3-6 个月）

| 目标 | 内容 |
|:------|:------|
| 会选股 | 能从板块扫描到个股筛选 |
| 会读表 | 三张财务报表（利润表 / 资产负债表 / 现金流量表） |
| 有系统 | 自己的交易系统（选股 → 买入 → 持有 → 卖出） |

**检验标准：** 连续 3 个月正收益，每笔交易有日志记录。
